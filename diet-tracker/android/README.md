# Android Release Notes

The Android project is generated by Capacitor from the Vue application.

## Local Build Environment

Use the temporary environment root requested for this task:

- JDK: `D:\项目\android-build\jdk\<jdk-directory>`
- Android SDK: `D:\项目\android-build\sdk`
- Gradle cache: `D:\项目\android-build\gradle-cache`

Set `JAVA_HOME`, `ANDROID_HOME`, `ANDROID_SDK_ROOT`, and `GRADLE_USER_HOME` to
these paths before invoking Gradle. Do not place an Android SDK, Gradle cache, or
keystore in the system drive for this build.

## Commands

From the project root:

```powershell
npm run cap:sync
npm run android:apk
```

The debug APK is expected at:

`android\app\build\outputs\apk\debug\app-debug.apk`

## Release Signing

Do not commit a keystore or passwords. Create a separate release keystore outside
the repository, configure it through local Gradle properties or CI secrets, then
build an AAB for application-store distribution.

## Privacy Defaults

`android:allowBackup` is disabled because the app stores diet and body records
locally. Users can explicitly export normal or encrypted backups from the app.
